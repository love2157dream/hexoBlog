# hexoBlog

``` bash
    # clean 清楚缓存文件和已经生成的静态文件
    hexo clean

    # generate 生成静态文件
    hexo generate  简写  hexo g

    # server 启动服务器，默认情况下，访问地址: http://localhost:4000/
    hexo server  简写 hexo s

    # deploy 部署网站
    hexo deploy 简写  hexo d

    命令简写
    hexo clean && hexo g && hexo s && hexo d
```
